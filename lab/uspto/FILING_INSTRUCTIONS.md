## Common pitfalls to avoid

1. **PO Box for owner domicile** — rejected. Use a street address.
2. **Mismatch between LLC name on USPTO filing vs. formation documents** — causes Office Action. Use the exact legal name.
3. **Filing as §1(a) actual use without a valid specimen** — causes Office Action. If unsure whether the mark is used in commerce as a source-identifier, default to §1(b) ITU.
4. **Custom-text identifications instead of ID Manual entries** — adds $200/class surcharge. All three drafts use approved IDs.
5. **Forgetting MFA on USPTO.gov account** — locks you out of filing. Set up tonight.
6. **Missing the 3-month Office Action response deadline** — application abandoned, fee lost. Calendar reminders are not optional.
7. **Treating the §1(b) verified statement as boilerplate** — it is a sworn statement under penalty of perjury. Be sure you actually have bona fide intent to use each mark as filed.

## If you decide to retain an attorney instead

If after reading `RISK_NOTES.md` you decide to retain a trademark attorney for one or more of these filings (recommended for MONEY PYTHON; optional for the others):

- Attorney can use these drafts as the basis for filing — they are not boilerplate to be discarded, but a starting point for an attorney to review and refine.
- Typical flat fee for a US-attorney-filed single-class trademark application: **$500-$1,500 attorney fees + $350 USPTO fee** per class.
- Online services (LegalZoom, Trademark Engine) are cheaper but you do not get the same substantive legal review.
- For the MONEY PYTHON filing in particular, a 30-minute consult with a trademark attorney before filing is likely worth it.

The drafts here are designed to be useful whether you file pro se or hand them to an attorney.
