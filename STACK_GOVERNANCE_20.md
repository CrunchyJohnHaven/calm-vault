# Calm Stack — 20 Governance + Standards Engineering Everests

**Cross-protocol governance, standards-track work, and adoption infrastructure.**

Where the four pillar route maps cover protocol-internal work, this document covers what happens at the *Calm Stack* level: standards, multi-party governance, external trust, and adoption. Stable IDs **CS-01 … CS-20**.

---

## Phase S-I — Cross-protocol governance (CS-01 – CS-08)

**CS-01** — Stack Versioning. Semver across the four pillars; one composed version (`calm-stack/v0`). *Effort:* S.
**CS-02** — Compatibility Matrix. Which pillar versions compose with which. *Effort:* M.
**CS-03** — Cross-Pillar Failure-Mode Reconciliation. Unified failure catalogue that maps pillar-specific FMs to common categories. *Effort:* M.
**CS-04** — Stack Quorum. Some operations (predicate-vocabulary changes, threat-model revisions) require quorum across pillar maintainers. *Effort:* M.
**CS-05** — Stack Open-Source Release Manifest. One canonical CHANGELOG across all repos. *Effort:* S.
**CS-06** — Bug Bounty Across Stack. Funded program; published payout schedule. *Effort:* M.
**CS-07** — Public Security Disclosure Policy. 90-day responsible-disclosure window. *Effort:* S.
**CS-08** — Public Roadmap. Living document; refreshed quarterly. *Effort:* S.

## Phase S-II — Standards track (CS-09 – CS-14)

**CS-09** — NIST AI Safety Institute Submission Bundle. Formal submission across all four protocols. *Effort:* L. *Prereq:* CS-05.
**CS-10** — IETF Draft for Calm Witness Wire Format. *Effort:* L.
**CS-11** — IETF Draft for Calm Pact Σ-Protocol. *Effort:* L.
**CS-12** — W3C VC Extension Draft for Calm Stack credentials. *Effort:* L.
**CS-13** — ISO Liaison. Information-security standards body engagement. *Effort:* M.
**CS-14** — Academic Paper. Peer-reviewed publication of the bank-teller-note primitive. *Effort:* XL.

## Phase S-III — Adoption infrastructure (CS-15 – CS-20)

**CS-15** — Counterparty Onboarding Wizard. New counterparty operators provision a verifier in under 30 minutes. *Effort:* M.
**CS-16** — Reference Counterparty Stack. Public open-source agent that runs the Calm Stack as a counterparty (for testing). *Effort:* L.
**CS-17** — Interop Test Suite. Public test vectors any implementation can run. *Effort:* L. *Prereq:* CS-15.
**CS-18** — Calm Stack Public Registry. A registry of operators + counterparties + their pinned VC hashes. *Effort:* L.
**CS-19** — Calm Stack Compliance Audit Firms. Named third-party audit firms with stated rates. *Effort:* M.
**CS-20** — Adoption Case Studies. Three public production case studies. *Effort:* L. *Prereq:* CS-15.
