Your AI agent should never hold your keys.

Calm Vault is a local, passphrase-protected credential broker. It hands your agents short-lived, scoped, revocable grants — never the underlying secret. One Python file. Apache 2.0.

# revoke anything, by grant / agent / credential
$ calm-vault revoke slack-bot

How it works

Three boring ideas, executed honestly.

Local root of trust

Master key derived from your passphrase via Scrypt. The vault is Fernet-encrypted at rest (AES-128-CBC + HMAC-SHA256). No service to call. No account.

Short-lived grants

Every grant is an HMAC-signed envelope: credential id, agent name, expiry, nonce. Default duration is five minutes. The agent never sees the master key.

Real revocation

Kill a grant by id, an agent by name, or a credential by name. Revocation is immediate and cascades on the next redemption.
